## 收藏按钮：修复 History 不变粉 + 扩展到搜索页
- 根因：`vod_id` 类型不一致——首页收藏用 API 返回的数字，播放历史用路由参数（字符串）写入；Dexie `.equals()` 严格比类型，导致 History 卡片 `isFavorite` 恒为 false，不变粉。
- `src/store/favorites.ts`：新增 `normId()` 统一把 `vod_id` 归一为字符串存取（`add`/`remove`/`toggle`/`isFavorite` 全部用 `normId`）；`isFavorite` 额外兼容旧的未归一数字记录。
- `src/components/VideoCard.tsx`：新增 `showFavorite` prop（只显示收藏、不显示删除）；vod 变体 actions 拆分为 删除(showActions) + 收藏(showActions||showFavorite)。
- `src/pages/Search.tsx`：两处搜索结果 VideoCard 加 `showFavorite`，使搜索页也能收藏（不误触删除）。
- 说明：Douban/Bangumi 卡片是元数据、链接到搜索、无单一可播放 vod，未加收藏（避免收藏后无法播放的半成品）；如需「想看」式收藏可另排期。
- 验证：tsc 0、vitest 29/29、build 成功。
